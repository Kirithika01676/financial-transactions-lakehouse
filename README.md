# Financial Transactions Data Pipeline

## 📌 Overview
This project demonstrates an end-to-end data engineering pipeline using Databricks and PySpark.

It processes raw financial data and transforms it into an analytics-ready dataset for business insights.

---

## 📂 Dataset
- customers.csv
- accounts.csv
- transactions.csv

---

## ⚙️ Pipeline Steps
1. Data ingestion into Databricks
2. Data cleaning (handling nulls & duplicates)
3. Data transformation using joins
4. Creation of final analytics dataset
5. Generating insights using PySpark

---

## 🛠️ Technologies Used
- Databricks
- PySpark
- Python
- GitHub

---

## 📊 Key Insights
- Identified top spending customers
- Analyzed spending across merchant categories
- Detected high-value and potentially risky transactions

---

## 🚀 Project Outcome
Built a scalable data pipeline that converts raw financial data into meaningful business insights.
